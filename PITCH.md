# Pitch

GridSense is a local data science project about electricity reliability in Addis Ababa. The problem is close to daily life: outages affect studying, internet use, small businesses, food storage, and work.

Instead of pretending that a model trained on synthetic data can solve a real infrastructure problem, I used real evidence first. The project combines Addis Ababa reliability baselines, public outage-related evidence, Ethiopia Enterprise Survey indicators, and a real community-collected outage dataset captured through a privacy-safe Streamlit form.

The notebook uses Kujenga ideas: regression, modelling, visual storytelling, responsible interpretation, and a clear conclusion. The project now includes a trained and evaluated outage-risk classifier on the real collected dataset, with baseline comparison and transparent metrics. It remains a research and decision-support tool, not an official utility prediction system.
